Basic Probability and Statistics

Expectation value#

The expectation value of a function g:R→Rg : \R \to \Rg:R→R of a univariate continuous random variable X∼p(x)X \sim p(x)X∼p(x) is given by

EX[g(x)]=∫Xg(x)p(x)dx\mathrm{E}_{\mathrm{X}}[g(x)] = \int_{\Chi} g(x) p(x) dxEX​[g(x)]=∫X​g(x)p(x)dx

In case of ggg is a function of discrete random variable X∼p(x)X \sim p(x)X∼p(x) is given by

EX[g(x)]=∑x∈Xg(x)p(x)\mathrm{E}_{\mathrm{X}}[g(x)] = \sum \limits_{x \in \Chi} g(x) p(x)EX​[g(x)]=x∈X∑​g(x)p(x)

For multivariate random variable, the expectation value is defined element wise,

EX[g(x)]=[EX1[g(x1)]EX2[g(x2)]⋮EXD[g(xD)]]∈RD\mathrm{E}_{\mathrm{X}}[g(x)] = \begin{bmatrix} \mathrm{E}_{\mathrm{X_1}} [g(x_1)] \\ \mathrm{E}_{\mathrm{X_2}}[g(x_2)] \\ \vdots \\ \mathrm{E}_{\mathrm {X_D}}[g(x_D)]\end{bmatrix} \in \R^DEX​[g(x)]=⎣⎡​EX1​​[g(x1​)]EX2​​[g(x2​)]⋮EXD​​[g(xD​)]​⎦⎤​∈RD

Correlation#

Uncorrelated: E(XY)=E(X)E(Y)\mathcal{E}(XY) = \mathcal{E}(X) \mathcal{E}(Y)E(XY)=E(X)E(Y), where E\mathcal{E}E denotes expectation values.

Positive correlation: E(XY)>E(X)E(Y)\mathcal{E}(XY) > \mathcal{E}(X) \mathcal{E}(Y)E(XY)>E(X)E(Y)

Negative correlation: E(XY)<E(X)E(Y)\mathcal{E}(XY) < \mathcal{E}(X) \mathcal{E}(Y)E(XY)<E(X)E(Y)

Measure of correlation: Covariance

Cov(X,Y)=E(XY)−E(X)E(X)Cov(X, Y) = \mathcal{E}(XY) - \mathcal{E}(X) \mathcal{E}(X)Cov(X,Y)=E(XY)−E(X)E(X)

Pearson's correlation coefficient:

E(XY)−E(X)E(X)σ(X)σ(Y)\frac{\mathcal{E}(XY) - \mathcal{E}(X) \mathcal{E}(X)}{\sigma(X) \sigma(Y)}σ(X)σ(Y)E(XY)−E(X)E(X)​

Range for Pearson's coefficient: [-1, 1].

Note: If two variables are uncorrelated, their covariance is 0. However, the reverse need not be true: two variables can dependent on each other and still can have zero covariance.

Bayes' rule#

Definition of conditional probability:

P(A∣B)=P(A∩B)P(B)P(A|B) = \frac{P(A \cap B)}{P(B)}P(A∣B)=P(B)P(A∩B)​, if P(B)≠0P(B) \neq 0P(B)=0

P(A∣B)P(A|B)P(A∣B) is probability of event AAA given that event BBB is true/occurring.

P(A∣B)=P(A)P(B∣A)P(B)P(A|B) = \frac{P(A) P(B|A)}{P(B)}P(A∣B)=P(B)P(A)P(B∣A)​
