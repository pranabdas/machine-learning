Ridge regression and Lasso

Ridge regression#

L(w,b)=∑i[yi−(w⋅xi+b)]2+λ∣w∣2L(w, b) = \sum_i [y_i - (w \cdot x_i + b)]^2 + \lambda |w|^2L(w,b)=i∑​[yi​−(w⋅xi​+b)]2+λ∣w∣2

Linear regression is a good choice when we have lots of training data. When we try to minimize LLL with higher value of of λ\lambdaλ, it will result in lower www's. Increasing λ\lambdaλ results in less weight for training data. In reality, we can vary λ\lambdaλ and choose a value for which the test error is minimum.

In case of Ridge regression also, we have a closed form solution:

w=(XTX+λI)−1(XTy)w = (X^T X + \lambda I)^{-1}(X^T y)w=(XTX+λI)−1(XTy)

Lasso#

Similar to Ridge regression:

L(w,b)=∑i[yi−(w⋅xi+b)]2+λ∣w∣L(w, b) = \sum_i [y_i - (w \cdot x_i + b)]^2 + \lambda |w|L(w,b)=i∑​[yi​−(w⋅xi​+b)]2+λ∣w∣

It has some advantages, like it produces sparse www matrix (lots of zero elements).
